Checksum Calculator

This was a project written in Java for my Security In Computing class. The objective of the program is to take a large text file and calculate a number that is unique to that file. For cyber security purposes, a file's number can be checked to "verify" it. If the file's checksum does not match the expected checksum, then the file has been modified.

Skills learned

  • Command line input for Java programs
  • Basic Checksum calculations and logic
